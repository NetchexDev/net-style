    Netchex Style Guide

    This guide provides guidelines for UI elements, colors, and text for Netchex web applications. The project is based on Bootstrap 5.0 (BS5). All BS5 helper classes and modifiers still apply. Some additional custom classes have also been added to this project and have been documented on their pages.
