# Netchex Styles
> Styles for Netchex

## NOTE: THIS REPO HAS BEEN MOVED TO AZURE DEVOPS AND IS NO LONGER MAINTAINED HERE

## Usage

Include the `/dist/net-style.min.css` or use a SASS compiler with `/scss/style.scss`


## Build

Build via npm:

```
$ npm run build
```


## Docs

Update the doc styles via:

```
$ npm run build-docs
```

## Fonts

Build the fonts:

```
$ npm run build-font
```
